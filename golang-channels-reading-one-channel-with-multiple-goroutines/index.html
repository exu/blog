Golang channels - Reading one channels with multiple goroutines

Reading from ticker by multiple goroutines is quite simple. We need iterate through it values using range.

package main

import "time"
import "fmt"

func main() {
	ticker := time.NewTicker(time.Millisecond * 500)

	go func() {
		for t := range ticker.C {
			fmt.Println("Ene", t)
		}
	}()

	go func() {
		for t := range ticker.C {
			fmt.Println("Due", t)
		}
	}()

	go func() {
		for t := range ticker.C {
			fmt.Println("Fake", t)
		}
	}()

	time.Sleep(time.Second * 4)
	ticker.Stop()
	fmt.Println("Ticker stopped")
}

Run it on play.golang.org

Code Summary

There are several important things in this example:

  • until we sleep we’ve had 3 goroutines, each of them try to read from our ticker channel, but only one in given period of time can do this,
  • it looks like there is something like queue (this should be confirmed! it’s only observation)
